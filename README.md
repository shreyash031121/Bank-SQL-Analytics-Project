# Bank Transaction & Fraud Analytics (MySQL)

This project analyzes bank transaction data using SQL to understand customer behavior and detect potential fraud patterns.

## Key Concepts Used
- Window Functions (SUM, RANK, NTILE)
- GROUP BY and HAVING
- Self Joins
- Date-based analysis
- Conditional aggregation

## Analysis Performed
- Running balance calculation
- High-value transaction detection
- Fraud patterns using time-based logic
- Inactive account identification
- Account ranking and risk scoring

## Tool Used
- MySQL
